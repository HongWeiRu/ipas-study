PART 01跨科共通術語

三科共用的關鍵名詞

這些名詞會在三科裡反覆出現，理解了它們，看任何一科都會順很多。每組標註它在哪些科目重點出現。

資料與品質類 S2 主導

結構化資料Structured Data明確欄位、固定型別的資料（資料庫表格）。來源：ERP、CRM、POS
半結構化資料Semi-structured具層次但欄位可變（JSON、XML）。來源：API、IoT、日誌
非結構化資料Unstructured無固定格式（文字、圖、音、影）。需特徵萃取
資料漂移Data Drift輸入資料分佈隨時間改變，模型效能下降
概念漂移Concept Drift資料背後 X 與 y 的關係改變，需重新訓練
資料標註Data Annotation為原始資料賦予標籤，作為監督學習的 Ground Truth
資料品質 5 指標完整性、一致性、準確性、即時性、唯一性
資料治理Data Governance政策、組織、流程、技術 + 文化的整合管理機制
資料血緣Data Lineage追蹤資料從來源到應用的完整流向；錯誤回溯關鍵
Data Lake / Lakehouse資料湖：原始格式儲存所有型態，Schema-on-Read；Lakehouse 結合湖的彈性 + 倉的治理

模型與訓練類 三科共用

監督式學習Supervised使用標註資料 (X, y) 學習映射。分類、迴歸
非監督式學習Unsupervised無標註資料，找潛在結構。分群、降維
強化式學習Reinforcement透過互動的試誤學習；最大化累積報酬
過擬合 / 欠擬合Over- / Under-fitting過：訓練好但測試差（死背）；欠：訓練測試都差（模型太簡單）
特徵工程Feature Engineering將原始資料轉為模型可用特徵，影響效能最大的環節
交叉驗證Cross-ValidationK-fold、Stratified、LOOCV、Repeated K-fold
超參數Hyperparameter訓練前需設定（學習率、樹深度），非由訓練學得
正則化RegularizationL1（Lasso）特徵選擇 / L2（Ridge）平滑權重 / Elastic Net / Dropout / Early Stopping
偏差-變異權衡Bias-Variance Tradeoff低複雜度高偏差；高複雜度高變異；找 Sweet Spot
批次大小Batch Size小批：震盪大易跳脫局部；大批：穩定但易陷局部

PART 02模型對照表

五大模型對照速查

考前最後一刻最有用的速查 — 給情境直接挑模型。

迴歸模型 S3

模型特性適用 / 限制
線性迴歸最簡單；解釋性強；五大假設（線性、誤差常態、變異齊一、獨立、無共線性）線性關係 + 需可解釋；對離群值敏感
Lasso（L1）權重絕對值加總；部分權重縮為零自動特徵選擇；高維資料
Ridge（L2）權重平方和；限制大小但不為零多重共線性嚴重時
SVR找 ε-tube；對離群值有抗性非線性 + 中小資料；計算成本高
決策樹迴歸非參數；無需常態假設；預測為分段常數非線性 + 混合特徵；易過擬合
隨機森林迴歸多棵樹平均；抗過擬合；可平行中到大型資料；可計算特徵重要性
Gradient Boosting序列修正誤差；精度高競賽常勝；訓練慢、超參多
XGBoost / LightGBM高效梯度提升；Level-wise vs Leaf-wise；支援稀疏與缺值業界首選；XGBoost 較不易過擬合，LightGBM 更快

分類模型 S3

模型特性適用 / 限制
邏輯迴歸sigmoid 映射到 [0,1] 機率；Log-Odds 線性假設二元分類 + 需解釋；金融、醫療
SVM最大化間隔；核函數處理非線性（RBF、Polynomial）中小資料 + 高維 + 邊界清晰
決策樹分類條件分支；Gini 不純度 / Information Gain解釋性強；易過擬合
隨機森林多樹投票；穩定且抗過擬合多數場景表現好
KNN最近 k 鄰居多數類別；懶惰學習小資料 + 特徵空間明顯；對量與維度敏感
樸素貝氏貝氏定理 + 特徵獨立假設文字分類（垃圾郵件、情感）；訓練快
XGBoost / LightGBM同迴歸版本；分類任務一樣強勢結構化資料 + 高精度需求

聚類與降維 S3

模型特性適用 / 限制
K-means分 K 個球狀群；需事先給 K；對初始中心敏感已知群數 + 群為球狀
DBSCAN基於密度；不需指定群數；自動標識離群點任意形狀群 + 有噪聲
階層式分群建立 Dendrogram；可探索層級結構無需指定群數；計算量大
GMM多個高斯分佈混合；軟分群可處理橢圓形群
PCA線性降維；最大化變異視覺化、雜訊過濾、加速建模
LDA監督式降維；最大化類間差異分類任務的降維首選
t-SNE非線性；保留局部結構視覺化專用；不適合建模預處理
UMAP類 t-SNE 但保留更多全域結構速度快、可解釋性較好
Autoencoder非線性降維；用神經網路重建深度結構需求、生成模型

深度學習架構 S3

架構特性主要應用
MLP全連接網路；攤平輸入；參數量大結構化資料；其他架構的後置分類器
CNN卷積 + 池化；權重共享；位置不變性影像辨識、工業檢測、自動駕駛
RNN / LSTM / GRU循環連接；序列記憶；LSTM 解決梯度消失NLP、語音、時序預測
Transformer純自注意力；全並行；位置編碼NLP、跨模態（BERT、GPT、T5、ViT）
GAN生成器 vs 判別器對抗訓練圖像生成（StyleGAN）；訓練不穩
VAE變分自編碼器；學潛在分佈異常偵測、潛在空間建模
Diffusion逐步去噪；高品質生成DALL·E、Stable Diffusion；當前圖像生成主流

CNN 經典模型演進

LeNet (1998)早期 CNN 鼻祖；手寫數字辨識
AlexNet (2012)深度學習里程碑；ReLU + Dropout；ImageNet 突破
VGG (2014)3×3 小卷積核堆疊；19 層
GoogLeNetInception 模塊；平行不同大小卷積核
ResNet (2015)殘差連接；可達 152 層；解決梯度消失
DenseNet每層與所有前層連接；特徵重用
MobileNet / EfficientNet輕量級；行動裝置與邊緣部署

優化器與激活函數

優化器特性適用
SGD最基本；計算簡單但震盪大小模型、淺層網路
MomentumSGD + 慣性；減少震盪深層網路、高曲率
Adagrad累積梯度平方；自適應學習率稀疏特徵（NLP）；學習率會持續衰減
RMSprop滑動平均修正 Adagrad序列建模（RNN）
AdamMomentum + RMSprop；萬用當前最常用；幾乎所有深度學習
激活函數特性使用層
ReLUmax(0, x)；稀疏激活；計算快；解決梯度消失隱藏層主力
Sigmoid壓縮到 [0, 1]；可作機率二元分類輸出層
Tanh壓縮到 [-1, 1]；中心對稱RNN 隱藏層常用
Softmax把向量轉為機率分佈（總和為 1）多類別分類輸出層

常見陷阱與口訣

PITFALL 01
α vs p 值
α 事前設定（判斷門檻）；p 值事後計算（觀察結果）。
p < α 拒絕 H₀；p ≥ α 無法拒絕（≠ 接受）
PITFALL 02
95% CI 的正確解讀
不是「母體平均有 95% 機率落在區間內」
而是「重複抽樣後，95% 的 CI 會涵蓋母體真值」
PITFALL 03
p 值膨脹
大數據下 p 值極敏感 — 差異 0.05% 也能算出 p < 0.0001。
對策：同時看效果量（effect size）+ 信賴區間
PITFALL 04
多重比較
同時做 20 次 α=0.05 檢定 → 出現一次假陽性 64%！
用 Bonferroni 校正（α / 檢定次數）
PITFALL 05
中位數 vs 平均數
極端值多 → 用中位數。
口訣：平均 > 中位 → 正偏；平均 < 中位 → 負偏
PITFALL 06
類別不平衡 + Accuracy
95% 樣本為負類 → 全預測為負仍 95% 準確率！
必須用 F1、AUC、Recall
PITFALL 07
L1 vs L2 正則化
L1 (Lasso)：絕對值 → 部分權重變零 → 自動特徵選擇
L2 (Ridge)：平方 → 權重變小但不為零 → 平滑
PITFALL 08
超參數 vs 模型參數
超參數需手動設定（學習率、Batch Size、層數）
模型參數由訓練學習（權重、偏差）
PITFALL 09
監督 vs 非監督
有 (X, y) 標籤 → 監督式（分類、迴歸）
只有 X 沒標籤 → 非監督式（分群、降維）
PITFALL 10
商業 vs 技術 vs 風險評估
商業看「值不值得」（ROI、效益）；技術看「做不做得到」（成熟度、資源）；風險看「會出什麼狀況」
PITFALL 11
BERT vs GPT
BERT：雙向 Encoder → 理解類任務（分類、QA、NER）
GPT：單向 Decoder → 生成類任務（對話、續寫）
PITFALL 12
生成式 vs 鑑別式 AI
生成式：學分佈 → 創造新內容（GPT、DALL·E）
鑑別式：學決策邊界 → 分類預測（SVM、ResNet）

考試當天提醒

選擇題答題技巧 ① 先看選項再看題目可能更快理解問題；② 遇到複雜情境，先圈出關鍵字（如「不平衡」、「即時」、「敏感資料」）；③ 不會的題目用刪去法排除明顯錯誤；④ 不要花太多時間在一題，先做有把握的；⑤ 留時間檢查 — 特別是「以下何者錯誤」「不適用」這類否定題。
常見出題類型 ① 名詞解釋型：「下列何者為 ___」
② 情境選擇型：「企業遇到 ___，最適合的方法是」
③ 對應辨識型：「下列哪一個屬於 ___ 範疇」
④ 否定排除型：「下列何者不是 ___」（要特別小心）
⑤ 方法比較型：「A 與 B 的差別是」
把這五種題型練熟，考試會非常順。
