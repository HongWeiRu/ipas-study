## 專案緣由

備考 iPAS 智慧人才公司的「中級 AI 應用規劃師」認證時，發現官方 PDF 學習指引內容紮實但**閱讀體驗較沈重**：
- 純文字段落多、圖示少
- 概念散在多個章節難以串連
- 沒有口訣與視覺化記憶輔助

於是決定把整套指引<b>重新組織</b>為網頁版教材 — 全文涵蓋、結構化呈現、輔以大量視覺化。

## 視覺設計特色

- 🎨 **magazine / 報紙風**排版（Noto Serif TC + Noto Sans TC + JetBrains Mono）
- 🌈 每科獨立**主色系**，視覺上一眼分辨在哪一科
- 📐 大量 **SVG 圖解**：NLP 四世代演進、Transformer 自注意力、知識圖譜流程、客服 pipeline、三層風險全圖…
- 🃏 **tcard / verb badge / 口訣**系統：例如用 6 個動詞「指 → 數 → 秤 → 串 → 學 → 懂」記憶 NLP 詞向量演進
- 💡 **CSS-only Tooltip**：縮寫名詞（MLM、NSP、BPE、SVD…）懸停可看完整定義
- 📚 **真實案例貫穿**：Amazon 履歷歧視、Apple Card 性別事件、ChatGPT 律師案…
- ⚙️ **純前端**：HTML + CSS + 內嵌 SVG，無需 build、無外部依賴、可離線使用
